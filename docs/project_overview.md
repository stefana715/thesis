# Project Overview

## Purpose
This repository supports a PhD thesis on low-data urban solar potential assessment in Changsha, China.

## Initial idea
The project is inspired by recent deep learning-based urban facade PV studies, but is adapted to lower data availability and open-source workflows.

## Initial workflow
1. Collect open-source geospatial and weather data
2. Build a baseline solar potential workflow
3. Create a structured dataset
4. Train and evaluate a machine learning model
5. Generate maps and thesis-ready figures
